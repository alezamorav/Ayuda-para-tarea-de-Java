Instrucciones makefile
----------------------
    1. Tener el terminal en la carpeta con las carpetas .vscode, bin, lib, src y los archivos makefile y README
    2. Poner el comando [make] o [make cowompile] en el terminal para compilar los archivos .java
    3. Para correr el juego se debe utilizar el comando [make ruwun]
    4. Para limpiar los archivos .class se debe utilizar el comando [make clean]

-------------------------------------
Instrucciones programa / Suposiciones
-------------------------------------
    Para continuar después de cada línea del juego se debe presionar la tecla Enter, no seguirá el programa si se presiona cualquier otra tecla.
    Las preguntas y/n son case insensitive.
    No se saboteará el funcionamiento del programa.
    En los menu se ingresará un input aceptado por éste (integer dentro de las opciones dadas)
    No se tratará de utilizar los métodos de manera que éstos no funcionen (saber que hay elementos nulos dentro de ellos, etc.)
    Las divisiones no exactas se redondean hacia abajo
    No se repetirán nombres de Persona, Edificio o Atracción
